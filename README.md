# Portfolio — 山原佑介

自己紹介ポートフォリオサイト

## 技術スタック

- HTML5
- CSS3
- JavaScript (Vanilla)

## ローカル確認

`index.html` をブラウザで直接開く。ビルドツール不要。

## GitHub Pages

設定後、`https://<username>.github.io/portfolio/` で公開予定。

## フォルダ構成

```
portfolio/
├── index.html
├── css/
│   ├── reset.css   — リセットCSS
│   └── style.css   — メインCSS
├── js/
│   └── main.js     — メインJS
├── img/            — 画像素材
└── docs/
    └── specs/      — 要件定義・設計ドキュメント
```
