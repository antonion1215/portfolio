# サービスページ 詳細設計（service.html）

作成日: 2026-06-22
ブランチ: `feature/sub-pages`

## 概要

提供サービス・料金プランを掲載するページ。**内容が未確定のため枠組みのみ**（プレースホルダ）。デザイン・共通ヘッダー/フッターは TOP と統一。

## 構成

1. **共通ヘッダー**（TOPと同一マークアップ。ナビは `index.html#xxx` へ）
2. **ページヘッダー（service_head）**: ラベル `SERVICE` ＋ タイトル「サービス」＋ 一言リード
3. **サービス内容（service_items）**: サービス項目を3枠のプレースホルダで表示（例: Webサイト制作 / LP制作 / 保守・運用）。各「準備中」
4. **料金プラン（service_plans）**: 料金表の枠組み（プラン名・価格を「準備中」プレースホルダ）。3プラン想定
5. **CTA**: 「お問い合わせはこちら」→ `index.html#contact`
6. **共通フッター**（TOPと同一）

## アニメーション

TOPと同方針（GSAP / ScrollTrigger）。各ブロックのスクロール連動フェードアップ。`main.js` の既存ロジックを共通利用。

## 実装メモ

- ページ固有クラスは `service_` プレフィックス（例: `service_head`, `service_items`, `service_plans`）
- 既存モジュール（`m_section-label` / `m_btn` / カード等）を再利用
- 新規CSSは `style.css` に `/* ---------- サービスページ ---------- */` セクションで追記
- 料金・サービス内容の実テキストは後で差し込む（プレースホルダにTODOコメント）
