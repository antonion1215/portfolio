# 実績一覧ページ 詳細設計（works.html）

作成日: 2026-06-22
ブランチ: `feature/sub-pages`

## 概要

制作実績を一覧表示するページ。**現時点では実績ゼロのため枠組みのみ**（「準備中」プレースホルダ）。実績が増えたらカードを差し替えていく。デザイン・共通ヘッダー/フッターは TOP と統一。

## 構成

1. **共通ヘッダー**（TOPと同一マークアップ）
   - ロゴ → `index.html` に戻る
   - ナビ各項目 → `index.html#about` のように TOP の該当セクションへ
   - スマホはハンバーガーメニュー（同仕様）
2. **ページヘッダー（page_head）**: ラベル `WORKS` ＋ タイトル「制作実績」＋ 一言リード
3. **実績グリッド（works_list）**: 「準備中」カードを6枠表示（スマホ1列 → PC3列）
4. **TOPへ戻る導線**: 「トップへ戻る」リンク
5. **共通フッター**（TOPと同一）

## アニメーション

TOPと同方針（GSAP / ScrollTrigger）。ページヘッダーのフェードアップ＋カードのスクロール連動stagger。`main.js` の既存ロジックを共通利用。

## 実装メモ

- ページ固有クラスは `works_` プレフィックス（例: `works_head`, `works_list`）
- 既存の `m_section-label` / `m_work-card` / `m_link-more` 等のモジュールを再利用
- 新規CSSは `style.css` に `/* ---------- 実績一覧ページ ---------- */` セクションで追記
